## 기온 비교 그래프 
<img src="https://github.com/hoheer/-SDOT_ACC/blob/main/image/%EA%B3%B5%EC%9B%90%EA%B3%BC%EB%8F%84%EC%8B%AC%EC%A7%80%EA%B0%84%EB%B9%84%EA%B5%90%EB%B6%84%EB%A6%AC.png" width="50%" height="40%" title="px(500)" alt="RubberDuck"></img>


21년 7월 한달간의 일별 평균을 가지고 그려 보았다.

도심에 비해 녹지에서 확실히 낮은 기온이 측정되었는데 특이한 점으론 기온이 낮으면 낮을수록 도심/녹지간 차이는 줄어든다는 점. 


문화비축기지의 온도 차이가 가장 적은 것은 센서의 위치때문으로 생각된다. 


문화비축기지의 센서 위치는 타 녹지와 달리 그늘 아래가 아닌 곳에 설치되어 있기 때문으로 보인다.
